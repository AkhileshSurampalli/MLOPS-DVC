1. Create a git repo and clone it in local.
2. Create code.py and add code to it.
3. Do a git add-commit-push before initializing dvc.
4. pip install dvc
5. Now do "mkdir S3"
6. Now we do "dvc remote add -d myremote S3"
7. Next "dvc add data/"
    Now it will ask to do: ("git rm -r --cached "data" and "git commit -m 'stop tracking data'")
    Because initially we were tracking data from git, we need to remove it
8. Again we do "dvc add data/" (creates data.dvc) then "git add .gitignore data.dvc"
9. Now do dvc commit and dvc push
10. Do a git add-commit-push to mark this stage as first version of data.
11. Again - - dvc commit and the dvc push
12. Then git add-commit-push (saving V2 at this point)
13. Check dvc/git status, everything should be upto date.
